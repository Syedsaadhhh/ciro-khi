## Recording Tips

1. **Pre-run each scenario** before recording to warm up the backend
2. **Clear previous traces** before each scenario for clean visuals
3. **Use cursor highlighting** (e.g., yellow circle) for important UI elements
4. **Keep mouse movements slow and deliberate** for readability
5. **Record audio separately** if possible for cleaner sound
6. **Add captions/subtitles** for accessibility
