## Recording Tips

1. **Open all files in tabs beforehand** for quick switching
2. **Use editor zoom (Ctrl+= or Cmd+=)** so code is readable at 1080p
3. **Highlight important lines** with cursor or editor selection
4. **Keep scrolling slow** — judges need time to read code
5. **Don't show terminal commands** — focus on files and workspace
6. **Avoid showing `.env` or credentials** — security
